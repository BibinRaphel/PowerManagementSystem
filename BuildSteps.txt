Installing modules
1. Install GCC - sudo apt install build-essential -y
2. Install sqlite3 - sudo apt install sqlite3 libsqlite3-dev
3. pip install flask

Compiling Code
1. mkdir .build
2. sqlite3 .build/energy.db < sql/energy_schema.sql
3. gcc logger/logger.c -lsqlite3 -o .build/logger -lcurl
4. gcc user_query/query_latest.c -o .build/query_latest -lsqlite3

Execution
1. python3 server/server.py
2. ./.build/logger
3. ./.build/query_latest
